#  Портфолио «Обо мне» — Junior Python

Консольное приложение на Python с текстовым меню, реализованное в рамках конкурса портфолио **CapEducation**.

---

```bash
python main.py
```

Требования: **Python 3.8+**, сторонние библиотеки не нужны.

---

## Структура проекта

```
portfolio/
├── main.py          # Точка входа, меню
├── portfolio.py     # Класс Portfolio (ООП)
├── utils.py         # Вспомогательные функции (рамки, валидация regex)
├── logger.py        # Логирование действий
├── logo.py          # Логотип через Turtle (бонус)
├── data.json        # Весь контент портфолио
└── README.md
```

---

##  Разделы меню

| # | Раздел |
|---|--------|
| 1 | О себе |
| 2 | Моя цель |
| 3 | Как я пришёл в IT |
| 4 | Мой ментор |
| 5 | Точка А → Точка Б |
| 6 | Хобби и интересы |
| 7 | Мои лучшие работы |
| 8 | GitHub |
| 9 | Поиск по портфолио |
| 10 | Логотип (Turtle) |
| 0 | Выход |

---

##  Особенности

- **ООП**: класс `Portfolio` хранит данные и выводит разделы
- **JSON**: весь контент вынесен в `data.json`, не захардкожен
- **Regex**: валидация ввода через `re.fullmatch`
- **Логирование**: все действия записываются в `portfolio.log`
- **Turtle**: бонусный логотип с анимацией
- **Поиск**: поиск ключевого слова по всем разделам
